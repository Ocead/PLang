General

Print help

:h

Prints a list of availabe commands with a brief description.

Parameters
(none)

Get/Set options

:p <option> <value>

Gets or sets an option of the command line.

Parameters
optionOption to set
value(opt) New value of the option

See the Command Line Options page for a list of available options.

Print current information

:i <ref>

Prints information about an entry in the corpus.

Parameters
ref(opt) Entry to print. If left blank, the current scope is printed out.
