Formatting options

output

Sets the assumed frontend connected to stdout

Allowed values
ansi Includes formatting through ANSI escape sequences
readline Includes formatting through ANSI escape sequences with additional markers for Readline
html Includes formatting through HTML tags and CSS stylesheets
doxygen Includes formatting through HTML tags and CSS stylesheets provided by Doxygen

enrich

Sets formatting to apply

Allowed values
plain Plain text without formatting
color Colored text
font Text with font modifications (bold, italic, etc.)
rich Text with color and font modifications

detail

Sets how PLang entries should be displayed in output

Allowed values
id_ref As reference through IDs
explicit_ref As reference without IDs
definition Equivalent PLang markup
full Equivalent PLang markup and related information

qualification

Sets how paths are output

Allowed values
unique Shortened paths where possible
full Always fully-qualified paths

indent

Sets how PLang markup is indented

Allowed values
compact Most compact notation
one_line Spaced one-line notation
left Multiline notation, left aligned
center Multiline notation, centered

Parser options

implicit

Whether entries may be declared implicitly

Allowed values
true
false

strict

Whether hints in class declarations are binding

Allowed values
true
false

topology

Sets the order for plot points.

Allowed values
svo Subject – Verb – Object
Example:
symbol.class[symbol] point.class : default_object ?object_class: object;
sov Subject – Object – Verb
Example:
symbol.class[symbol] : default_object ?object_class: object point.class;
vso Verb – Subject – Object
Example:
point.class symbol.class[symbol] : default_object ?object_class: object;
vos Verb – Object – Subject
Example:
point.class : default_object ?object_class: object symbol.class[symbol];
ovs Object – Verb – Subject
Example:
: default_object ?object_class: object point.class symbol.class[symbol];
osv Object – Subject – Verb
Example:
: default_object ?object_class: object symbol.class[symbol] point.class;
