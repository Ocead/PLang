# Contributing

**Thank you very much** for considering contributing to this project!<br/>
Your input is very valuable.

> ℹ You don't need knowledge or a background in programming to contribute!

## Issues <small>_(code & everything but)_</small>

> ℹ You can make an issue of anything, really. <br/>Feature requests, additions to the codebase, bug reports, comments, feedback, questions;<br/>
> all is welcome.

1. Navigate to the [<u>☉ Issues</u>](https://github.com/Ocead/PLang/issues) tab and create a [New Issue].
2. Choose appropriate labels on the sidebar
3. Fill in title and comment as precise as you feel necessary (I don't require much form)
4. If you have additional resources you want to share, link or upload them
5. Submit the new issue

## Pull requests <small>_(code)_</small>

> ℹ Please first discuss the change you wish to make via issue or email,
> with [me](https://github.com/Ocead) before making changes.

1. Fork this repository
2. Make the changes you want to contribute on the forked repository<br/>
   (Please also update any documents and documentation accordingly)
3. File a pull request

In each case, I'll get back to you as soon as I can.
