plang::format Class Reference

Sets options for text output. More...

#include <detail.hpp>

Detailed Description

Sets options for text output.

Objects of this type may be passed to corpus' print* functions to include various formatting.

Supported display backends.

Enumerator
ANSI 

For ANSI terminals.

Output will contain ANSI escape sequences like \033[1;31m.

READLINE 

For use within GNU Readline.

HTML 

For usage in HTML pages.

Output will contain ANSI escape sequences like <span style="color: red;"></span>

DOXYGEN 

Like output::HTML, but with different tags.

This output mode may override styles
