Detailed Description

The main interface for interacting with a corpus.

This class contains multiple mixins for various parts of the corpus named like <PART>_manager.
See the "Public Member Functions inherited from plang::detail::<PART>_manager" sections of the documentation for the methods each mixin provides.

Note
Check the friends of a class to see which mixin in responsible for handling it
