Resolves a path to persisted nodes to the most viable candidate.

Parameters
candidatesList of candidates as returned by path_manager::partially_resolve

Direct children or parents will be preferred in case of ambiguity otherwise.
Setting this to the id of the root node will treat path as fully-qualified.

◆ wipe()

void plang::detail::path_manager::wipe ( )

Wipes the whole path tree.

Warning
Calling this function will delete all content from the corpus!

Resets the database into a state equal to being just created through the plang::corpus constructor. Thus, the corpus may be used afterwards just as before the call.
